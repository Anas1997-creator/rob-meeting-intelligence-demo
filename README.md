# Rob Meeting Intelligence Demo

Visual prototype for Rob's Meeting Intelligence dashboard.

## What this is
A Railway-ready demo app using static sample data from Rob's Robbinsville transcript.

## What is not connected yet
- No real n8n workflow
- No Claude API
- No Airtable integration
- No Trello integration
- No Plaud or Granola integration

## Local run
```bash
npm start
```
Open: http://localhost:3000

## Railway deploy
1. Upload this folder to GitHub.
2. Create a new Railway project from the GitHub repo.
3. Railway should run `npm start`.
4. Open the generated Railway URL.

## Demo login
Any values work. Click "Open demo dashboard".
